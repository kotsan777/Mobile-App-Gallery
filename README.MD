# Mobile App Galery

## Frameworks: 
- [SDWebImage](https://github.com/SDWebImage/SDWebImage)

## Architecture pattern:
- MVP

## Modules:
- WelcomeModule
- AuthModule
- GalleryModule
- PhotoModule

## Short description:
- This Application working with [VK API](https://dev.vk.com/api/getting-started)
- Log-In with your VK account
- After Log-in you will see gallery with photo
- After select any photo you will see this photo
- You can also share or save to camera roll this photo
